팩트풀니스 데이터 리터러시 수업

학생들이 “세상은 점점 나빠지고 있다” 같은 직관을 바로 믿지 않고, 최신 공공데이터로 자신의 상식을 업데이트하는 수업용 전용 페이지입니다.

수업 핵심 질문

내가 알고 있는 세계의 모습은 최신 데이터와 얼마나 다를까?

페이지를 열면 최신 CSV를 자동으로 불러와 기본 그래프를 보여 줍니다. 학생은 먼저 예상값을 적고, 예상값과 비교하기로 실제값과의 차이를 확인합니다.

배우는 역량
  • 예상과 관측값 구분
  • 출처와 최신성 확인
  • 국가/세계 평균 비교
  • 그래프 과장 읽기
  • 근거 있는 설명 쓰기
데이터를 불러오는 중입니다…

1단계. 데이터 보기 전 예상하기

그래프를 보기 전에 아래 질문에 먼저 답해도 좋고, 빈칸으로 둔 채 실제값만 확인해도 됩니다.

데이터를 불러온 뒤 예상-실제-차이를 표시합니다.

2단계. 최신 지표 비교하기

선택한 지표의 국가별 가장 최근 값을 비교합니다. 아래 선택 상자와 최신 비교 그래프·추세 그래프가 함께 바뀝니다.

3단계. 시간에 따른 변화 보기

팩트풀니스 수업에서는 “현재 스냅샷”보다 “변화 방향”이 중요합니다. 같은 지표를 시간축으로 보면 오래된 상식이 왜 남아 있는지 토론할 수 있습니다.

데이터 출처와 한계

  • 출처: World Bank 지표를 갱신 스크립트로 수집해 만든 factfulness_global_indicators.csv
  • 페이지에서 불러오는 경로: ../data/factfulness_global_indicators.csv
  • 지표별 최신 연도는 국가마다 다를 수 있습니다. 그래프의 hover/막대 라벨에서 연도를 함께 확인하세요.
  • 이 페이지는 “정답 맞히기”보다 예상과 데이터가 달라지는 이유를 해석하는 수업용 자료입니다.

4단계. 토론 질문

  1. 내 예상과 실제 데이터가 가장 크게 달랐던 지표는 무엇인가?
  2. 나는 왜 그렇게 예상했을까? 뉴스, 교과서, 경험, 편견 중 무엇의 영향이 컸을까?
  3. 세계 평균만 보면 놓치는 차이는 무엇인가? 국가별 비교가 필요한 이유는?
  4. 그래프 축을 바꾸면 인상이 어떻게 달라지는가?
  5. 최신 데이터로 설명문을 다시 쓴다면 어떤 문장이 더 정확한가?

수업 산출물 예시

  • “내 상식 업데이트 카드” 1장: 예상값, 실제값, 차이, 배운 점
  • 국가별 지표 비교 인포그래픽
  • 오래된 통념을 바로잡는 1분 발표
  • 데이터 출처·연도·한계를 포함한 짧은 보고서
