SVG 지도 이미지 시각화 예제

수업용으로 바로 이해하기 쉬운 SVG 지도/카토그램 예제입니다. 실제 보고서용 경계는 국토교통부 행정구역 경계 SHP, 서울 열린데이터광장/주소기반산업지원서비스 경계, 또는 행정동 GeoJSON을 SVG로 변환해 같은 방식으로 칠하면 됩니다.

1) 한국 광역시도별 시각화

시도를 클릭해 보세요.

아이디어: 인구, 미세먼지 평균, 학교 수, 지역 축제 수, 생물종 관측 수 등을 시도별로 합산해 색으로 표현합니다.

2) 서울 구별 시각화

서울 25개 구를 SVG 느낌의 카토그램 타일로 표현했습니다. 실제 구 경계 SVG를 써도 id와 fill을 바꾸는 구조는 같습니다.

구를 클릭해 보세요.
