# Foot Traffic / Floating Population APIs

유동인구 데이터는 상권 분석, 시간대별 패턴, 날씨/이벤트와의 관계 분석에 좋습니다. 다만 개인정보 비식별화와 이용 약관을 함께 가르치는 것이 중요합니다.

## 서울 생활인구 / 상권 데이터 후보
- 주제: 시간대별 생활인구, 상권별 추정 유동인구, 성별/연령대별 패턴
- 인증 유형: 서울 열린데이터광장 또는 공공데이터포털 제공 API/파일별 상이
- 공식 포털: https://data.seoul.go.kr/
- 교육 포인트:
  - 시간대별 히트맵
  - 행정동/상권 단위 집계
  - 날씨·대중교통·매출 데이터와 결합
  - 개인정보 보호와 비식별 집계 데이터 이해
- Streamlit 적합도: 높음. 지도/히트맵/시간 슬라이더에 적합
- Pico W 적합도: 낮음. 데이터가 크고 분석 중심이라 서버/노트북 권장

## 통신/카드 기반 유동인구 후보
- 주제: 상권 방문자, 시간대별 혼잡, 관광지 방문 추정
- 인증 유형: 제공기관별 상이. 무료 샘플/공공 개방 데이터 위주 확인 필요
- 교육 포인트: 표본 편향, 개인정보, 집계 단위, 정책 분석
- 비고: 실제 API보다 CSV/파일 데이터가 많을 수 있음
