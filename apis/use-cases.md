# Education Use Cases

## Streamlit 프로젝트

### 1. 오늘의 도시 데이터 대시보드
- API: Open-Meteo + Nominatim + 서울 따릉이
- 학습: API 호출, 좌표, 지도, 현재값 metric
- 확장: 장소 검색 후 해당 좌표 날씨 표시

### 2. 영화 검색/추천 미니앱
- API: TMDB 또는 OMDb
- 학습: 검색 폼, JSON 정규화, 이미지 카드, 장르 필터
- 확장: 사용자가 좋아하는 영화 3개 입력 → 비슷한 장르 추천

### 3. 주식 시계열 분석 앱
- API: Stooq 또는 Alpha Vantage
- 학습: CSV/JSON 로딩, OHLCV, 이동평균, 수익률
- 확장: 종목 2개 상관관계 비교

### 4. 한국 공공데이터 분석 앱
- API: 기상청 단기예보, 금융위원회 주식시세, 한국은행 ECOS
- 학습: 서비스키, 공공데이터포털 활용신청, 코드표, 날짜 파라미터
- 확장: 날씨와 따릉이 이용량 관계 분석

## Pico W 프로젝트

### 1. 미니 날씨 스테이션
- API: Open-Meteo
- 출력: 시리얼, OLED, LED 색상
- 장점: 키 불필요, 응답 작음

### 2. 위치 검색 후 좌표 표시
- API: Nominatim
- 출력: 장소명/위도/경도
- 주의: 반복 호출 금지, 캐싱 권장

### 3. 현재 주가 티커 표시기
- API: Stooq CSV 또는 Finnhub quote
- 출력: OLED에 심볼/현재가 표시
- 주의: 투자 조언 문구 금지, 호출 간격 길게

### 4. 영화 오늘의 추천 표시기
- API: TMDB/OMDb
- 출력: 제목/연도/평점
- 주의: 키 저장은 `secrets.py`로 분리
