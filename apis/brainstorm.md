# API Brainstorm for Data Science Education

수업용으로 좋은 API는 “학생이 질문을 만들기 쉬운 데이터”가 좋습니다. 아래는 자체 브레인스토밍 기준입니다.

## 우선순위 기준

1. 무료로 시작 가능
2. JSON/CSV 응답이 단순함
3. 지도·시간·순위·비교 중 하나 이상을 다룰 수 있음
4. Streamlit 대시보드로 시각화하기 좋음

## 추천 주제 축

### 대중교통
- 질문 예시: “우리 학교 근처 버스는 몇 분 뒤 오나?”, “자전거가 많이 남는 동네는?”
- API 후보: 서울 버스도착정보, 서울 따릉이, GTFS/대중교통 정류소 데이터
- 수업 포인트: 실시간 데이터, 위치, 지도, 혼잡도, 대기시간

### 인구
- 질문 예시: “한국 인구는 언제부터 줄었나?”, “도시별 고령화는 어떻게 다른가?”
- API 후보: World Bank, KOSIS, 행정안전부 주민등록 인구통계
- 수업 포인트: 장기 시계열, 지역 비교, 인구 피라미드, 결측치

### 야구
- 질문 예시: “개막일 팀별 득점 분포는?”, “선수 기록으로 MVP 후보를 예측할 수 있나?”
- API 후보: MLB Stats API, pybaseball, Retrosheet/Lahman DB, KBO 공공/파일 데이터 후보
- 수업 포인트: 스포츠 통계, 순위, 승률, 선수별 누적 기록

### 유동인구
- 질문 예시: “비 오는 날 유동인구가 줄까?”, “상권별 시간대 패턴은?”
- API 후보: 서울 생활인구/상권 데이터, 통신사/공공 유동인구 개방 데이터 후보
- 수업 포인트: 시간대별 패턴, 히트맵, 상권 분석, 개인정보 비식별화 토론

### 기상
- 질문 예시: “기온과 따릉이 이용량은 관련이 있나?”, “내 위치의 초단기예보는?”
- API 후보: Open-Meteo, 기상청 단기예보, AirKorea 대기오염 후보
- 수업 포인트: 시계열, 예보와 관측 차이, 격자 좌표, 센서 프로젝트

### 영화
- 질문 예시: “장르별 평점 차이가 있나?”, “좋아하는 영화 기반 추천은?”
- API 후보: TMDB, OMDb
- 수업 포인트: 검색, 추천, 텍스트/카테고리 데이터, 이미지 카드 UI

### 지도
- 질문 예시: “주소를 좌표로 바꾸고 주변 시설을 지도에 표시하면?”
- API 후보: Nominatim, Kakao Local/Map, Google Maps
- 수업 포인트: 지오코딩, 거리 계산, 지도 시각화, API 정책

### 주식/경제
- 질문 예시: “이동평균 전략은 어떤 한계가 있나?”, “금리와 환율은 같이 움직이나?”
- API 후보: Stooq, Alpha Vantage, 한국은행 ECOS, 금융위원회 주식시세
- 수업 포인트: OHLCV, 수익률, 변동성, 상관관계, 투자 조언 금지

## Streamlit 프로젝트 묶음 추천

1. “날씨 × 따릉이 × 지도” 도시 데이터 대시보드
2. “인구 × 경제지표” 한국/세계 장기 변화 분석
3. “영화 검색과 추천” 취향 기반 미니앱
4. “야구 스코어보드” 경기 일정/결과/순위 시각화
5. “주식 이동평균 실험실” CSV/JSON 기반 시계열 분석
6. “유동인구와 상권” 시간대별 히트맵
