# Transit APIs

## 서울특별시 버스도착정보조회 서비스
- 주제: 서울 버스 정류소별 도착 예정 정보, 혼잡도/잔여좌석 일부 정보
- 인증 유형: 공공데이터포털 회원가입 + 무료 활용신청 + 서비스키 필요
- 공식 문서: https://www.data.go.kr/data/15000314/openapi.do
- 문서 확인 내용: 정류소 번호, 첫 번째/두 번째 도착예정 버스 메시지, 도착예정시간(초), 노선ID, 차량유형, 우회여부 등 제공
- 교육 포인트:
  - 실시간/준실시간 대중교통 데이터
  - 정류소 ID와 노선 ID 조인
  - 도착 시간 예측값 시각화
  - 혼잡도 코드 매핑
- Streamlit 적합도: 높음. 정류소 검색 + 도착 버스 카드 UI

### 활용신청/키 발급 방법
1. https://www.data.go.kr/ 회원가입/로그인
2. API 페이지에서 활용신청
3. 자동승인 여부 확인 후 일반 인증키(Encoding/Decoding)를 복사
4. 요청 URL의 `serviceKey`에 키 입력
5. 학생 실습에서는 교사용 키를 서버 환경변수로 관리 권장

## 서울 열린데이터광장 따릉이 실시간 대여정보
- 주제: 서울 공공자전거 대여소 위치, 거치대 수, 현재 자전거 수
- 인증 유형: 샘플키 `sample`로 일부 테스트 가능. 정식 사용은 무료 키 신청 권장
- 테스트 URL: http://openapi.seoul.go.kr:8088/sample/json/bikeList/1/5/
- 교육 포인트: 지도, 좌표, 숫자 변환, 정렬, 지역별 집계
- Streamlit 적합도: 매우 높음

## GTFS / 정적 교통 데이터 후보
- 주제: 정류소, 노선, 시간표, 운행 경로
- 인증 유형: 제공처별 상이. 많은 GTFS 파일은 다운로드 기반
- 교육 포인트: 여러 CSV 테이블 조인, 경로 시각화, 시간표 분석
- 비고: 국내 제공처는 기관별로 달라 추가 조사 필요
