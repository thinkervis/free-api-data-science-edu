# 기여 안내 (사람 버전)

이 레포는 **초·중·고 정보 교육을 위한 무료 데이터 과학 교육 자료**를 함께 만드는 프로젝트입니다. 선생님, 학생, 개발자, 시민 누구나 기여할 수 있습니다.

## 어떤 기여를 환영하나요?

- 수업에 쓰기 좋은 무료 API/CSV 데이터 추천
- 최근 5년치 CSV 자동 갱신 스크립트 추가
- Streamlit 예제 개선
- 데이터별 설명 문서, 수업 활동지, 질문 예시 추가
- 오타 수정, 링크 검증, 라이선스/출처 확인
- 초등/중등/고등 난이도별 활동 아이디어 제안

## 데이터 추가 기준

기본 조건:

1. 무료로 시작 가능해야 합니다.
2. 공식 문서 또는 공식 데이터 페이지가 있어야 합니다.
3. GitHub Pages에서 직접 테스트 가능한 링크가 있어야 합니다.
4. 기본 CSV는 가능한 한 최근 5년치로 제공합니다.
5. 원하면 전체 기간을 받을 수 있도록 자동화 코드 옵션을 둡니다.
6. 개인정보, 민감정보, 저작권 문제가 없어야 합니다.

추천 데이터:

- 키 없이 바로 테스트 가능
- JSON/CSV 구조가 단순함
- 초·중·고 학생이 질문을 만들기 쉬움
- 지도, 시간, 순위, 비교, 예측 중 하나 이상 가능

## Pull Request 체크리스트

PR을 보내기 전에 확인해주세요.

- [ ] 공식 문서 URL을 추가했습니다.
- [ ] 직접 테스트 가능한 URL을 추가했습니다.
- [ ] `scripts/update_datasets.py`에 최근 5년치 CSV 생성 로직을 추가했습니다. (해당하는 경우)
- [ ] `python3 scripts/update_datasets.py --scope recent5` 실행 성공
- [ ] `python3 scripts/generate_pages.py` 실행 성공
- [ ] `python3 scripts/validate_datasets.py` 실행 성공
- [ ] 데이터별 Streamlit 기본 코드가 GitHub Pages에 노출됩니다.
- [ ] 출처와 라이선스/이용 조건을 문서에 적었습니다.

## 제안만 하고 싶다면

GitHub Issue에 아래 형식으로 남겨주세요.

```md
## 데이터/API 이름

## 공식 문서 URL

## 왜 초중고 정보 교육에 좋은가?

- [ ] 무료 키 필요
- [ ] 가입 필요
- [ ] 결제 설정 가능성 있음

## 직접 테스트 URL

## 수업 아이디어
```
